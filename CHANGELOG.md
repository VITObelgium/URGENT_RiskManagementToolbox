# Change Log
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)

## [0.2.0] - 2026-02-18
### Added
- Initial release of the URGENT Risk Management Toolbox
- Support for OpenDarts Connector
- Well design service for vertical, J shape, S shape, and horizontal wells
- Optimization parameters section for configuring optimization settings
- PSO optimization engine with support of single and multi-objective optimization
- Configuration example for setting up the toolbox
